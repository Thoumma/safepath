[EN] Home screen — the main tab after login. A large red SOS button in the centre (press to send an emergency alert), an orange prompt card asking the user to complete their profile so the embassy can identify them during an SOS, and a setup-status checklist (encrypted passport stored / trusted contacts added) that updates live. Bottom navigation: Home, Trusted contacts, About, Report. Android emulator view (1440×3040).
