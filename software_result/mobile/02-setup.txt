[EN] Password setup screen — create the account with two passwords: the real password (opens the normal vault) and an optional fake/duress password (opens an empty decoy vault and silently fires an SOS when someone forces the user to unlock the app). Both are stored only as salted PBKDF2 hashes. Android emulator view (1440×3040).
